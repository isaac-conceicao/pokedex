# PokedexApp
Welcome, fellow coach! On Pokedex Web App you'll be able to explore a list with 1st Gen pokémons and their details. My main goal on this project is to have a little fun while practicing the very basics of front-end web development (CSS, HTML, JavaScript).

## Features

- [x] Load Pokémon data from [PokeAPI](https://pokeapi.co/)
- [x] List Pokémons
- [x] Style by Type
- [x] Pagination
- [x] Load More
- [x] Add Basic Details
- [x] Pokemon as Cards
- [x] Flip-Card effect
- [x] Blur effect
- [ ] Pokemon Details Page
- [ ] Filter Pokemon
- [ ] Favorite Pokemon

## Como Usar

1. Clone the rep;
2. Open `index.html` file on your browser.
